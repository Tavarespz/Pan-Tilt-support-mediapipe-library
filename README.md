# Pan-Tilt-support-mediapipe-library

This project, presented by Guilherme Tavares and Davi Sarttorreto, consists of a pan-tilt camera mount using a 3D-printed base with two servomotors: one for the X-axis (pan) and another for the Y-axis (tilt). The mount is controlled by an Arduino, which converts hash-format data into binary to determine axis movement; Python code, utilizing the Mediapipe library for camera-based facial tracking, performs the processing required to meet the project's objectives.

Este projeto, apresentado por Guilherme Tavares e Davi Sarttorreto, consiste em um suporte *pan-tilt* para câmera, utilizando uma base impressa em 3D com dois servomotores: um para o eixo X (*pan*) e outro para o eixo Y (*tilt*). O suporte é controlado por um Arduino, que converte dados de formato *hash* para binário para determinar o movimento dos eixos; um código em Python, utilizando a biblioteca Mediapipe para rastreamento facial via câmera, realiza o processamento necessário para atender aos objetivos do projeto.


Dependencies Command line for download:
